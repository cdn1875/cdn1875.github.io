# Rawrardl Reference

This page is a simple reference hub for information related to
raw manga and Japanese digital publications.

It does not host or distribute any files.

For more organized information, you can check:
https://rawrardl.info/

---

In addition, for those interested in Japanese music-related information,
there is a separate reference site that focuses on music titles and releases:

https://jpmusicdl.site/

Both sites are intended to help users browse and organize publicly available
information more efficiently.
